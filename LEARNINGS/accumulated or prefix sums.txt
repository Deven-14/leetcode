if we have an array [23,2,4,7,7]
and if we want to know the longest arry with whose elements sum up to 'k' or whose elements are divisible by 'k'
let k = 6
then 
prefix sum = [23,25,29,36,43]
with % k
prefix sum % k = [5, 1, 5, 0, 1]

Here - start at idx0 (5) to idx2 (5)
This can only happen when we add 'k' or a multiple of 'k'
eg. hash = (num + prime) % prime - to avoid the certain issues
similarly (5) idx0 to (5) idx2 represents the same thing, i.e. [2, 4] = 6
so, length = idx2 - idx0 = 2

similarly (1) idx1 to 1 idx4, [4, 7, 7] = 18
so, length = idx4 - idx1 = 3
